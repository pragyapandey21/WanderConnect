# 🌍 WanderConnect

**Explore • Connect • Journey Together**

WanderConnect is a full-stack MERN travel companion platform that helps travelers discover destinations, connect with like-minded people, share travel experiences, and communicate in real time. The platform combines social networking with travel planning to make exploring new places more interactive and community-driven.

---

## ✨ Features

### 🔐 Authentication
- Secure user registration and login
- JWT-based authentication
- Google Sign-In support
- Protected routes

### 🗺️ Destination Explorer
- Interactive India map
- Explore popular destinations
- Search destinations
- Destination details page with travel information

### 👥 Find Travel Companions
- Discover travelers by destination
- Filter travelers based on travel style
- Follow/Unfollow users
- Open-to-chat status

### 💬 Real-Time Chat
- One-to-one messaging
- Group travel chats
- Instant messaging using Socket.IO
- Pinned messages
- Online status

### 📸 Travel Posts
- Upload travel photos
- Add captions and locations
- Tag destinations
- Like and comment on posts
- Share travel experiences

### 👤 User Profile
- Edit profile
- Upload profile picture
- Add travel interests
- Add travel style
- View followers & following
- Track visited destinations

### 📍 Destination Details
- Traveler statistics
- Hotels
- Events
- Active travel groups
- Ratings & reviews

### 📱 Responsive UI
- Mobile-first design
- Modern glassmorphism interface
- Smooth animations
- Dark neon theme

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/WanderConnect.git
```

## Install Frontend

```bash
cd client
npm install
npm run dev
```

## Install Backend

```bash
cd server
npm install
npm start
```
